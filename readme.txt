بکاپ خودکار دیتابیس و فایل‌ها + آپلود به Files.ir

== Description ==

افزونه بکاپ کامل وردپرس با امکانات:

* بکاپ دیتابیس (mysqldump/PHP)
* بکاپ فایل‌ها (ZIP/TAR.GZ)
* آپلود خودکار به Files.ir
* زمان‌بندی روزانه/هفتگی
* مدیریت Retention
* Worker URL

== Installation ==

1. آپلود فایل ZIP
2. فعال‌سازی
3. تنظیم API از Files.ir
4. تنظیم زمان‌بندی
