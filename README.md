# Files.ir WordPress Backup

افزونه بکاپ خودکار وردپرس با آپلود مستقیم به Files.ir

## ویژگی‌ها

✅ بکاپ خودکار دیتابیس (mysqldump یا PHP)  
✅ بکاپ فایل‌ها (ZIP یا TAR.GZ)  
✅ آپلود به Files.ir با 3 روش (Simple/Stream/Chunked)  
✅ زمان‌بندی روزانه/هفتگی  
✅ Worker URL برای cron سرور  
✅ مدیریت Retention  
✅ UI مدرن با تب‌بندی  

## نصب

1. فایل ZIP را آپلود کنید
2. به تنظیمات → Files.ir Backup بروید
3. تنظیمات API را وارد کنید
4. زمان‌بندی را تنظیم کنید
